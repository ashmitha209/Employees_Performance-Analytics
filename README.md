# Employee Performance Analytics using SQL

## Project Overview

This project analyzes employee performance, employee satisfaction, project profitability, client details, and attrition using SQL.

## Dataset Details

The dataset contains:

- Employee ID
- Employee Name
- Department
- Designation
- Experience
- Technology Stack
- Project Details
- Client Details
- Budget and Actual Cost
- Employee Satisfaction Score
- Attrition Flag
- Performance Rating

## Tools Used

- MySQL Workbench
- SQL
- Excel
- Power BI

## Analysis Performed

### Employee Analysis

- Top-performing employees
- Least-performing employees
- Overtime analysis
- Most experienced employees

### Department Analysis

- Average department performance
- Employee satisfaction
- Attrition analysis

### Technology Analysis

- Performance by technology stack
- Satisfaction by technology
- Technology-wise attrition

### Project Analysis

- Most profitable projects
- Budget overrun analysis
- Extra working hours

### Client Analysis

- Country-wise projects
- Client profitability
- Client-wise project count

## SQL Concepts Used

- WHERE
- GROUP BY
- ORDER BY
- Aggregate Functions
- Window Functions
- Subqueries
